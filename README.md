# 🌍 Vision Forge

> **AI-Powered Smart Municipal Complaint Management System**

Vision Forge is a modern web application that enables citizens to report civic issues while allowing municipal authorities to efficiently monitor, assign, and resolve complaints using AI-powered image classification and analytics.

# ⚙ Installation

Clone the repository

```bash
git clone https://github.com/krishna/VisionForge.git
```

Go inside project

```bash
cd VisionForge
```

Install dependencies

```bash
npm install
```

Run development server

```bash
npm run dev
```

---

# 🌐 Environment Variables

Create a `.env` file.

```env
VITE_API_URL=http://localhost:5000/api
VITE_GOOGLE_MAP_API_KEY=YOUR_GOOGLE_MAP_KEY
```
